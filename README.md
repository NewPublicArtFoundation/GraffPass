# Public Art

Internet's largest database of geotagged graffiti.

### Unicorn stuff

In RAILS_ROOT, run:

  `unicorn_rails`

Unicorn will bind to all interfaces on TCP port 8080 by default.
You may use the +--listen/-l+ switch to bind to a different
address:port or a UNIX socket.
